# SMARTATTEND
-SmartAttend is a web-based smart attendance system that uses QR code and location verification to record student attendance efficiently and prevent fraud.

## FEATURES
- Student registration and login
- QR code-based attendance marking
- Real-time time tracking
- Location verification to prevent proxy attendance
- Easy-to-use interface for students and lecturers

# TOOLS USED
-Git
-html
-css
-javascript
-github

# GOALS
-To provide a fast, secure, and reliable attendance system that can be used in schools and institutions worldwide.
